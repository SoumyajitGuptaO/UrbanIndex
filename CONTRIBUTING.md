# Contributing

Thanks for contributing to Urbis.

**Before you start**
- Ensure your changes build with CMake.
- Run tests with `ctest --test-dir build`.

**Code style**
- C11 required.
- Keep functions small and focused.
- Prefer explicit error handling over implicit behavior.

**Docs**
- Update `README.md` or `api/README.md` when behavior changes.

**PR checklist**
- Tests pass.
- No unnecessary files are added.
- Public APIs are documented.
